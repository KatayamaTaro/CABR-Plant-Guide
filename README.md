# CABR-Plant-Guide

Heavy vibe coded html flashcard set for CABR native vegetation. Invasive flashcards to come. Includes invasive species as well. Currently lists 145 plants!


User guide:

1) download the html and .json file somewhere convenient.
2) Open html file on an internet browser like chrome or edge.
3) Import the .json file using the import .json file button in the html.
4) STUDY!
